Overview

Despite its name, @ks89/angular-modal-gallery is more than for modal galleries, because I'm introducing new features every major release. In fact, It's composed by 3 main parts:

  • MODAL GALLERY: available injecting the ModalGalleryService in your component
  • PLAIN GALLERY: available via <ks-plain-gallery></ks-plain-gallery>
  • CAROUSEL: available via <ks-carousel></ks-carousel>

MODAL GALLERY

It's the core part of this project and display a modal window with full screen images, buttons, the current image and optionally, also navigation dots and previews


  1. Current image
  2. Side previews
  3. Upper buttons
  4. Dots
  5. Previews
  6. Background (semi-transparent)
  7. Modal gallery (the main component)

For more info check Modal gallery documentation


PLAIN GALLERY

It contains a list of clickable thumbnails to open modal gallery. Layouts are configurable and users could also use a custom implementation to draw everything they want.


For more info check Plain gallery documentation


CAROUSEL

It contains a list of scrollable images (full width or fixed width) also with previews. By default images scroll automatically. Users can configure this component with many parameters, also they can project HTML content into the current carousel image.


For more info check Carousel documentation
